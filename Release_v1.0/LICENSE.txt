INSIGHTSEARCH - ULTRA FAST FILE SEARCH AND INDEXING TOOL

This program allows you to:
- Index and search through thousands of files instantly
- Support for multiple file formats (PDF, Office, Images, Text files)
- Advanced filename and content search capabilities
- Save and load search indexes for faster future searches
- Dark/Light theme support

System Requirements:
- Windows 7 or later
- Minimum 2GB RAM
- 100MB free disk space
- Optional: PDF support libraries for enhanced functionality
